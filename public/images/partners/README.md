# Logos des Partenaires

Placez les logos des partenaires dans ce dossier avec les noms suivants :

- `italiancharms.jpg` - Logo d'Italian Charms
- `rosalove.jpg` - Logo de Rosa Love
- `coffret.jpg` - Logo du Coffret à Montre

**Format recommandé :** JPG avec fond transparent si possible
**Dimensions recommandées :** 300x300px (carré)
**Poids max :** 200KB par logo

Les logos seront automatiquement arrondis et affichés avec les effets de surbrillance.
Les coins seront arrondis à 30 degrés automatiquement.
