# Logo Nexus Circle

Placez le logo dans le dossier `public/` :

## Fichiers requis :

- `logo.svg` - Logo en format SVG (PRIORITAIRE)
- `logo.png` - Logo en format PNG (fallback si SVG manquant)

## Spécifications :

**Format SVG (recommandé) :**
- Format vectoriel
- Fond transparent
- Couleur : blanc (#FFFFFF) ou or (#D4AF37)
- Le SVG doit être optimisé et propre

**Format PNG (fallback) :**
- Dimensions : 200x200px minimum
- Fond transparent (PNG-24)
- Haute résolution (2x pour Retina)

## Utilisation :

Le logo apparaît à 3 endroits :
1. **Header** : En haut à gauche avec le texte "Nexus Circle."
2. **Footer** : En bas avec le texte "Nexus Circle."
3. **Chatbot** : Avatar du bot dans la fenêtre de chat (rond, fond noir)

Le logo doit être simple, lisible et professionnel.
